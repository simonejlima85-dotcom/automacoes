# 🚀 Automações - V4 Company

Repositório de automações e processos criados por **Simone Lima** na V4 Company.

Aqui documentamos e versionamos as automações para facilitar a consulta, manutenção e compartilhamento de conhecimento.

## 📂 Estrutura

```
workflows/
├── relatorios/           → Automações de relatórios automáticos
├── imagens-listas/       → Extração de contatos via imagem + Telegram
└── (outras categorias no futuro)
```

## 📋 Índice de Automações

| Categoria          | Descrição                                      | Arquivo principal                     |
|--------------------|--------------------------------------------------|---------------------------------------|
| **Relatórios**     | Relatório semanal automático de campanhas     | `relatorios/relatorio-v4-semanal.json` |
| **Imagens & Listas** | Extração de contatos de imagens via Telegram + GPT | `imagens-listas/imagens-listas-telegram.json` |

## Como usar

1. Clone o repositório
2. Importe o arquivo `.json` desejado no seu n8n
3. Configure as credenciais e variáveis necessárias (estão indicadas nos READMEs de cada pasta)
